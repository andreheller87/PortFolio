Lista de tarefas.

Um dos projetos que desenvolvi foi um aplicativo de lista de tarefas (To-Do List) utilizando HTML, CSS e JavaScript. Este aplicativo permite que os usuários adicionem e gerenciem suas tarefas de forma simples e intuitiva. Ao criar esse projeto, pude aprimorar minhas habilidades em desenvolvimento front-end. Aprendi a utilizar eventos para capturar ações do usuário, como adicionar, editar e remover tarefas. Também implementei recursos como pesquisa e filtragem para facilitar a navegação e organização das tarefas. Utilizei o armazenamento local (localStorage) para salvar as tarefas do usuário, permitindo que elas persistam mesmo após o fechamento do aplicativo.
