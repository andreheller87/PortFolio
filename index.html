Sistema Stock Safe.

Estou atualmente envolvido no desenvolvimento de um Sistema de Estoque utilizando Java, que servirá como meu Trabalho de Conclusão de Curso (TCC) no programa Entra21. Este projeto está centrado na "Stock Safe", uma empresa fictícia especializada em oferecer soluções inovadoras de gestão de estoque para pequenas empresas. Ao seu término, o sistema contará com recursos avançados, como rastreamento em tempo real, alertas de níveis baixos de estoque, gerenciamento de prazos de validade, histórico detalhado de movimentação de estoque e relatórios personalizados.

Xadrez.

Projeto de Jogo de Xadrez em Java
Descrição: Durante o meu curso de Java da Udemy, tive a oportunidade de mergulhar no mundo da programação orientada a objetos e aplicar meus conhecimentos em um projeto prático emocionante: um jogo de xadrez implementado em Java. Interface de usuário simples e intuitiva, permitindo que os jogadores interajam facilmente com o jogo através do Git Bash.
